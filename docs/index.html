Capabilities

What You Get

Six core features designed for developers who demand total visibility into their Claude Code workflow.

Live Session Monitoring

Watch Claude Code transcripts unfold in real time. Follow mode auto-scrolls to the latest output. Cycle through subagent conversations with a single keypress.

Team & Agent Tracking

Monitor multi-agent teams, their assigned tasks, inbox messages, and cross-agent coordination. Drill down from teams to members to individual task details.

Smart Todo Management

Track todos across all your Claude Code sessions in one unified view. See what every agent is working on, what's done, and what's pending.

Git at a Glance

See staged, unstaged, and untracked files at a glance. View full diffs inline without running a single git command. Stay on top of every change.

Plan Viewer

Browse execution plans and markdown documents without leaving your terminal. Full scrollable content rendered in your TUI with syntax-aware formatting.

One-Command Launch

assoc launch opens Windows Terminal with Claude Code on the left and The Associate on the right. Zero setup, instant side-by-side workflow.

Getting Started

Three Steps. That's It.

1

Install

cargo install assoc

Or download the prebuilt binary from the releases page. Single executable, no runtime dependencies.

2

Launch

assoc launch

Run it from your project directory. Windows Terminal opens with Claude Code and The Associate side by side.

3

Monitor

Tab / j / k / h / l

Claude Code on the left, The Associate on the right. Navigate tabs with keyboard shortcuts. Everything updates in real time.
